# CSS Sidebar Menu

A simple sidebar menu project built using HTML and CSS.
This project uses a checkbox-based toggle system to open and close the sidebar without JavaScript.

## Features

* Sidebar open and close functionality
* Built using only HTML and CSS
* Font Awesome icons
* Hover effects
* Full-screen background image
* Smooth sidebar transition

## Tech Stack

* HTML
* CSS
* Font Awesome

## What I Learned

* How to structure a basic HTML page
* How to create a sidebar menu using CSS
* How to use the checkbox hack for toggling UI
* How to add icons using Font Awesome
* How to use positioning, hover effects, and transitions in CSS
